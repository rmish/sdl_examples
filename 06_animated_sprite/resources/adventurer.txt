Файл adventurer.png содержит 8 видов анимации для персонажа, смотрящего вправо, 
и 8 для персонажа, смотрялего влево. Размер одного кадра - 32х32. 

Каждый тип анимации расположен в строку (всего 16 строк).
Для разных видов анимации отрисовано несколько фаз движения: от 4 до 13, поэтому при работе 
из программы нужно иметь список максимальных фаз для каждого вида анимации.
Для обращения к одному кадру анимации нужно будет вычислить координаты его левого верхнего угла, 
что можно сделать по формуле: x=32*anim_phase y=32*anim_type, где anim_type - вид анимации 
от 0 до 15, anim_phase - текущий кадр от 0 до максимального для данного типа.
